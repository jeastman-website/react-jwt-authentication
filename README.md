# Ejemplo siemple de autenticación en React empleando JWT


``` bash
git clone https://github.com/jeastman-website/react-jwt-authentication.git
```

Instalar los paquetes
``` bash
npm i
```

Ejecutar
``` bash
npm run start
```

Visualizar en el navegador entrando la url:
```
http://localhost:3000
```
